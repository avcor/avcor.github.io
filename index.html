Mor Blood

Application build for meeting blood requirements. It helps you to find places to donate and setup appointments. It was considered for deploying in chhattisgarh, under chhattisgarh govt. Capable of storing data of millions of Indian, and filtering over it, to provide brisk results.

Build powerful searches such as

  • Auto Search- In which app does all the work, & user does'nt have to do anything. It comes up with 10 nearby user of 40Km geofencing.
  • Radial Search- Allows user to set geographical radius, in which app will find donors and return list to user
  • Name Search- Enter the name of place or enter pincode, boom and you will find the large list of donor
